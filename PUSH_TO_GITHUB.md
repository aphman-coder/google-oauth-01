# Push to GitHub - Commands

After creating your repository on GitHub, run these commands:

```bash
# Replace REPO_NAME with your actual repository name
git remote add origin https://github.com/aphman-coder/REPO_NAME.git
git branch -M main
git push -u origin main
```

Example if your repo is named "google-oauth-login":
```bash
git remote add origin https://github.com/aphman-coder/google-oauth-login.git
git branch -M main
git push -u origin main
```
